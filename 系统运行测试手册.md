## 系统运行测试
按照以下步骤测试：

1. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

2. **初始化数据库**
   ```bash
   flask db init
   flask db migrate -m "initial tables"
   flask db upgrade
   ```

3. **运行系统**
   ```bash
   python run.py
   ```

4. **访问系统**
   打开浏览器访问：`http://localhost:5000/auth/login`
   - 管理员账号：`admin` / `123456`
   - 学生账号：使用管理员添加的学生学号登录，默认密码为学号
   - 教师账号：使用管理员添加的教师工号登录，默认密码为工号
