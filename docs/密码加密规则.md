# 加密规则
**MD5 32位小写加密**
明文 `123456` → MD5结果：`e10adc3949ba59abbe56e057f20f883e`

## 1. 加密逻辑
1. 接收原始明文密码
2. 直接进行**MD5哈希加密**
3. 转为**全小写32位字符串**存入数据库

## 2. Python 实现代码
```python
import hashlib

def md5_pwd(pwd):
    return hashlib.md5(pwd.encode('utf-8')).hexdigest().lower()

print(md5_pwd("123456"))
# 输出：e10adc3949ba59abbe56e057f20f883e
```

## 3. 登录校验逻辑
用户输入密码 → 同样MD5加密 → 和数据库密文对比，一致则登录成功。
