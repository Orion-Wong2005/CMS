# 课程管理系统 使用说明书
**版本**：V2.0（2026年5月最新版）  
**技术栈**：Flask 3.1.0 + MySQL 8.0 + Bootstrap 5.3  
**适用角色**：系统管理员、教师、学生

---

## 一、系统概述
本系统是专为高校课程管理设计的数字化平台，实现了学生、教师、课程、选课、成绩、排课的全流程信息化管理，支持多角色权限隔离，操作简单直观，适合课程设计和中小型院校使用。

### 1.1 核心功能
| 角色 | 主要功能 |
|------|----------|
| **管理员** | 学生管理、教师管理、课程管理、排课管理、系统数据统计 |
| **教师** | 查看授课课程、管理学生名单、单个/批量录入成绩、导出成绩表、查看个人课表 |
| **学生** | 选课/退课、查看已选课程、查询成绩与绩点、查看个人课表 |

### 1.2 运行环境
- **浏览器**：Chrome 90+、Edge 90+、Firefox 88+
- **分辨率**：推荐 1920×1080 及以上
- **服务器**：Windows 10+/Linux（本地部署）或云服务器（公网访问）

### 2.1 系统访问

### 2.2 默认账号
| 角色 | 用户名 | 初始密码 | 说明 |
|------|--------|----------|------|

## 三、管理员操作指南
### 3.1 登录系统
1. 打开系统登录页面
2. 输入管理员用户名和密码
3. 点击"登录"按钮，进入管理员控制台

### 3.2 学生管理
#### 3.2.1 添加学生
1. 点击顶部导航栏"学生管理"
2. 点击右上角"添加学生"按钮
3. 填写学生信息（学号、姓名为必填项）
4. 点击"保存"按钮
> 说明：添加学生时系统会自动创建登录账号，**用户名=学号，默认密码=学号**

#### 3.2.2 编辑学生
1. 在学生列表中找到需要修改的学生
2. 点击右侧"编辑"按钮
3. 修改学生信息（学号不可修改）
4. 点击"保存"按钮

#### 3.2.3 删除学生
1. 在学生列表中找到需要删除的学生
2. 点击右侧"删除"按钮
3. 在弹出的确认框中点击"确定"
> 警告：删除学生将同时删除其选课记录和成绩记录，且无法恢复

### 3.3 教师管理
操作流程与学生管理完全一致：
1. 点击顶部导航栏"教师管理"
2. 进行添加、编辑、删除操作
> 说明：添加教师时系统会自动创建登录账号，**用户名=工号，默认密码=工号**

### 3.4 课程管理
#### 3.4.1 添加课程
1. 点击顶部导航栏"课程管理"
2. 点击右上角"添加课程"按钮
3. 填写课程信息（课程编号、名称、学分、学时为必填项）
4. 选择授课教师和开课学期
5. 设置课程容量（默认60人）
6. 点击"保存"按钮

#### 3.4.2 编辑/删除课程
- 编辑：点击课程右侧"编辑"按钮，修改信息后保存
- 删除：点击课程右侧"删除"按钮，确认后删除
> 警告：删除课程将同时删除所有学生的选课记录和成绩记录

### 3.5 排课管理
#### 3.5.1 添加排课
1. 点击顶部导航栏"排课管理"
2. 点击右上角"添加排课"按钮
3. 选择课程、星期、开始节次、结束节次和教室
4. 点击"保存"按钮
> 系统会自动检查**教室冲突**和**教师时间冲突**，冲突时会提示错误信息

#### 3.5.2 编辑/删除排课
- 编辑：点击排课右侧"编辑"按钮，修改信息后保存
- 删除：点击排课右侧"删除"按钮，确认后删除

## 四、教师操作指南
### 4.1 登录系统
1. 打开系统登录页面
2. 输入教师工号和密码（初始密码为工号）
3. 点击"登录"按钮，进入教师首页

### 4.2 查看我的课程
1. 点击顶部导航栏"我的课程"
2. 查看所有自己教授的课程列表
3. 列表显示课程名称、学分、学生人数和待录入成绩人数

### 4.3 查看课程学生名单
1. 在"我的课程"列表中，点击课程右侧"查看学生"按钮
2. 查看该课程的所有选课学生名单
3. 列表显示学生学号、姓名、专业、班级和成绩状态

### 4.4 成绩录入
#### 4.4.1 单个录入成绩
1. 在学生名单中，找到需要录入成绩的学生
2. 点击右侧"录入"或"修改"按钮
3. 输入成绩（0-100之间，支持0.5分小数）和备注
4. 点击"保存"按钮

#### 4.4.2 批量录入成绩
1. 在学生名单页面，点击右上角"批量录入成绩"按钮
2. 在表格中一次性输入所有学生的成绩和备注
3. 点击"保存所有成绩"按钮

#### 4.4.3 导出成绩表
1. 在学生名单页面，点击右上角"导出成绩表"按钮
2. 系统会自动生成Excel文件并下载
3. Excel文件包含学号、姓名、专业、班级、成绩和备注信息

### 4.5 查看我的课表
1. 点击顶部导航栏"我的课表"
2. 查看以周为单位的个人课表
3. 课表显示课程名称和上课教室

## 五、学生操作指南
### 5.1 登录系统
1. 打开系统登录页面
2. 输入学生学号和密码（初始密码为学号）
3. 点击"登录"按钮，进入学生首页

### 5.2 选课中心
1. 点击顶部导航栏"选课中心"
2. 查看所有可选课程列表
3. 列表显示课程名称、学分、授课教师、学期和容量/已选人数
4. 对于未满且未选的课程，点击右侧"选课"按钮
5. 在弹出的确认框中点击"确定"
> 说明：
> - 已选课程显示"已选"按钮，不可重复选课
> - 已满课程显示"已满"按钮，无法选课
> - 选课成功后会自动创建成绩记录

### 5.3 我的课程
1. 点击顶部导航栏"我的课程"
2. 查看所有已选课程列表
3. 列表显示课程名称、学分、授课教师、选课时间和成绩
4. 对于未出成绩的课程，点击右侧"退课"按钮可以退课
> 警告：已出成绩的课程无法退课

### 5.4 我的成绩
1. 点击顶部导航栏"我的成绩"
2. 查看所有已出成绩的课程
3. 页面顶部显示已修学分和平均绩点(GPA)
4. 列表显示课程名称、学分、成绩、绩点和备注

### 5.5 我的课表
1. 点击顶部导航栏"我的课表"
2. 查看以周为单位的个人课表
3. 课表显示课程名称、授课教师和上课教室

## 六、通用功能操作
### 6.1 修改密码
1. 点击右上角用户名下拉菜单
2. 选择"修改密码"
3. 输入旧密码、新密码和确认新密码
4. 点击"确认修改"按钮
5. 修改成功后系统会自动退出，需要使用新密码重新登录
> 要求：新密码长度不少于6位

### 6.2 修改个人信息
1. 点击右上角用户名下拉菜单
2. 选择"修改个人信息"
3. 修改手机号和邮箱（学号/工号、姓名等关键信息不可修改）
4. 点击"保存修改"按钮

### 6.3 退出登录
1. 点击右上角用户名下拉菜单
2. 选择"退出登录"
3. 系统会返回到登录页面

## 七、常见问题解答
### Q1：登录时提示"用户名或密码错误"
- 检查用户名和密码是否输入正确，注意区分大小写
- 学生用户名是学号，教师用户名是工号
- 初始密码与用户名相同，如果修改过请使用修改后的密码
- 如果忘记密码，请联系管理员重置

### Q2：选课失败，提示"课程容量已满"
- 该课程选课人数已达到上限，无法继续选课
- 可以选择其他未满的课程，或者等待有人退课后再尝试

### Q3：选课失败，提示"您已选过该课程"
- 每个学生只能选修同一门课程一次
- 可以在"我的课程"中查看已选课程

### Q4：无法退课
- 已出成绩的课程不允许退课
- 请联系管理员处理特殊情况

### Q5：成绩录入后无法修改
- 教师可以随时修改自己课程的学生成绩
- 如果无法修改，请检查是否登录了正确的教师账号

### Q6：课表中没有显示课程
- 确认已经成功选修了该课程
- 确认管理员已经为该课程安排了排课

## 八、系统维护说明
### 8.1 数据库备份
定期备份数据库，防止数据丢失：
```bash
# Windows本地备份
mysqldump -u root -p course_management_system > backup_YYYYMMDD.sql

# Linux服务器备份
mysqldump -u cms_user -p course_management_system > backup_YYYYMMDD.sql
```

### 8.2 数据库恢复
如果数据损坏，可以使用备份文件恢复：
```bash
# Windows本地恢复
mysql -u root -p course_management_system < backup_YYYYMMDD.sql

# Linux服务器恢复
mysql -u cms_user -p course_management_system < backup_YYYYMMDD.sql
```

### 8.3 日常维护
- 定期修改管理员密码，确保系统安全
- 定期清理无用的学生、教师和课程数据
- 定期备份数据库，建议每周备份一次
- 及时更新系统依赖包，修复安全漏洞
